# Application Submission Checklist

Date: 2026-06-10

Use this checklist right before submitting an application or sending the
project to a teacher, reviewer, or program contact.

## Repository State

- [ ] Repository link opens correctly
- [ ] Latest public release is `v0.1.5`
- [ ] Latest public release notes are visible
- [ ] `main` branch CI is green
- [ ] `Reviewer smoke proof` workflow is green
- [ ] `Contributor self-check` workflow is green
- [ ] `Community entry proof` workflow is green
- [ ] `Feedback triage proof` workflow is green
- [ ] `Submission review proof` workflow is green
- [ ] `OSS readiness proof` workflow is green
- [ ] Open issues are still focused on the remaining real-world gaps only

## Positioning Discipline

- [ ] I describe the project as an early-stage open source scaffold
- [ ] I do not claim real GPU benchmark results
- [ ] I do not claim independent public adoption
- [ ] I do not claim external validation from maintainer-controlled accounts
- [ ] I use the phrase `strong early-stage maintainer package with public reproducibility proof` when I need one short framing line

## Best Evidence To Mention

- [ ] public releases through `v0.1.5`
- [ ] passing CPU-only CI
- [ ] reviewer smoke proof workflow
- [ ] contributor self-check workflow
- [ ] community-entry proof workflow
- [ ] feedback-triage proof workflow
- [ ] submission-review proof workflow
- [ ] OSS-readiness proof workflow
- [ ] reproducibility evidence bundle validator
- [ ] GPU metrics preparation
- [ ] dry-validatable vLLM/L40S smoke profile
- [ ] result-submission example bundle
- [ ] result-review checklist
- [ ] issue chooser / contributor entry path

## Honest Remaining Gap

- [ ] I mention that the remaining gap is one real outside signal:
  - independent public tester feedback, or
  - one real L40S/vLLM artifact bundle

## Final Sanity Check

- [ ] My wording matches the public repo state
- [ ] My wording does not overclaim beyond the public evidence
- [ ] My wording is short enough for the actual form or message I am sending
