## Counting Rules

- Count only public comments, issues, or PRs from real external accounts.
- Do not count maintainer comments, automated comments, or private messages.
- Do not count stars or forks as tester feedback.
- Do not count feedback as benchmark validation unless it includes the required
  artifact evidence.
