# Limitations

- No real GPU or model-server results are included yet.
- The first benchmark client uses non-streaming requests, so time to first token
  is not measured in real mode yet.
- Dry-run numbers are synthetic and exist only to test the pipeline.
- Token counts are config-level targets, not tokenizer-verified counts.
- Published results should not include private cluster paths, hostnames,
  usernames, job IDs, API keys, or internal data.
